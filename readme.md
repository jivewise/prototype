# Prototype

## Install

Clone repository and run:

```sh
$ npm install
```

## Requirements

node 5+

## Development

```sh
$ npm start
```

Go to [http://localhost:3001](http://localhost:3001) and [http://localhost:3001/admin](http://localhost:3001/admin) to see the interface.
